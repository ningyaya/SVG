基于SVG的手势密码锁

360前端星计划大作业：实现一个移动网页，允许用户设置手势密码和验证手势密码。要求：

  • 设置密码 ：两次输入密码设置；密码长度不少于5位；密码记录在本地 localStorage。
  • 验证密码 ：验证输入是否为本地 localStorage 内存储的密码。

思路

  • 通过doSVG()函数将解锁九宫格画好，初始化密码为12369，需要先设置密码；
  • 选择设置密码时，setTime用来记录设置次数，setTime = 1时，将临时密码存入变量temp2Pass，需要再次输入密码来确认，当两次密码一致时，设置成功，将密码存入localStorage；不一致时，重新进入等待状态。
  • 选择验证密码时，从localStorage中取出密码，判断tempPass与passWord是否一致。

相关函数

  • doSVG()：

    在SVG(#nineSVG)里循环append九个circle，得到九宫格；
    给circle的touchstart和touchend事件绑定监听，当circle被按下时，开始画线并记录起初点index，存入变量tempPass；
    当触摸结束时，根据option的值是设置密码(setGesture)或验证密码(checkGesture)，执行相应的逻辑；
    给#nineSVG的touchmove事件绑定监听，画线，并将途经点存入变量tempPass；

  • cursorPoint(x, y)：
    通过pt.matrixTransform(svg.getScreenCTM().inverse())，实现将触摸点位置信息转换成svg位置信息。

  • inCircle(x, y)：
    实现判断当前触摸点是否在circle内，并返回圆的index值。
